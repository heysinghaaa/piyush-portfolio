# Piyush Singh — Portfolio

Personal portfolio site for **Piyush Singh**, a frontend developer and AI training contributor based in Jaipur, India. Built with Next.js and React.

## Stack

- [Next.js](https://nextjs.org/) 16
- React 19
- TypeScript

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start the dev server     |
| `npm run build` | Create a production build |
| `npm start`     | Serve the production build |
| `npm run lint`  | Run ESLint               |

## Project structure

- `src/app` — App Router pages and global styles
- `src/components` — UI pieces (contact form, typewriter heading, scroll effects)
- `src/data` — Portfolio content (profile, experience, projects, skills)
- `public` — Static assets (portrait, resume)
